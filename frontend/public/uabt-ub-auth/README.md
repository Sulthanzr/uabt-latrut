# UABT UB — Auth (Login & Register)

Standalone halaman autentikasi UABT UB (Unit Aktivitas Bulutangkis Universitas Brawijaya).

## Isi
- `login.html` — halaman login
- `register.html` — halaman daftar akun
- `auth.css` — styling tema gradasi hijau-kuning
- `auth.js` — toggle password
- `logo.png` — logo UABT UB

## Pakai
Buka `login.html` di browser, atau host folder ini sebagai static site terpisah (Netlify, Vercel, GitHub Pages).

## Deploy ke repo GitHub baru
Folder ini bisa dipindah ke repository baru sendiri (mis. `uabt-ub-auth`) — tidak punya dependency ke folder lain.
