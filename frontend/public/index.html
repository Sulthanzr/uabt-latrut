UABT LATRUT

Sistem latihan bulutangkis UABT UB untuk pemain dan admin: register akun, login, bergabung sesi, antrean match, generate match, dan monitoring sesi latihan.

Login Masuk ke akun pemain atau admin UABT UB. Register Daftar akun pemain baru untuk ikut sesi latihan. Dashboard Pemain Pantau profil, sesi aktif, statistik main, dan riwayat match. Admin Panel Kelola sesi, antrean pemain, generate match, dan export data.

Unit Aktivitas Bulutangkis Universitas Brawijaya.
